# Experiments

This directory centralizes investigation design, execution assets, and result summaries.

## Structure

- `optimal-setup/`: umbrella experiment for converging on the final analysis configuration.

Each sub-experiment follows the same pattern:

1. `DESIGN.md`: question, rationale, preflight, and scope.
2. `configs/` and `scripts/`: execution assets.
3. `results.md`: artifact pointers and conclusions.
