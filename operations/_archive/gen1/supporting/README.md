# Supporting Analyses

These studies validate, stress-test, or contextualize the cornerstone decisions in `optimal-setup/`.

- `methodology-audit/`: checks whether the analytical machinery behaves as assumed.
- `factorial/`: studies design-factor interactions.
- `4protein-comparison/`: compares the conservative 4-protein core against the broader consensus panel.
- `phase2-consensus/`: fresh-seed validation of the consensus panel.
- `permutation-testing/`: significance confirmation.

None of these directories replaces the main decision chain; they support confidence in it.
